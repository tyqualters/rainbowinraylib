# Taste the Rainbow with raylib

Controllable Rainbow using raylib and raygui

![rainbow.gif](rainbow.gif)

- [raylib](https://github.com/raysan5/raylib)
- [raygui](https://github.com/raysan5/raygui) -- INCLUDED for raylib 5.0
